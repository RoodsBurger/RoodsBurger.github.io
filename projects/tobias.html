Overview

Tobias is a quadrupedal robot project that integrates CAD design, physics simulation, and reinforcement learning to achieve stable walking gaits and straight-line locomotion. By combining advanced RL techniques with detailed physical modeling, we've created a robust platform for exploring autonomous robot movement and control.

Key Features

  • • End-to-end integration pipeline from CAD design to physical deployment using Fusion 360 and PyBullet
  • • Advanced reinforcement learning implementation comparing DQN and SAC algorithms for optimal gait learning
  • • Custom Controller Class for precise joint movement control and physical constraints management
  • • Successful sim-to-real transfer through both closed-loop control and action replay methods

Technical Details

Learning System

  • • SAC with entropy-based exploration and automatic temperature tuning
  • • 28D state space capturing position, orientation, velocities, and joint angles
  • • Continuous 8D action space controlling joint angles (2 per leg)
  • • Multi-component reward system with stability and efficiency terms
  • • 1M step training with 512 batch size and 1M buffer capacity

Architecture & Implementation

  • • Dual Actor-Critic networks with 256-neuron hidden layers and ReLU activation
  • • Custom Controller Class handling joint constraints and gear ratios
  • • PyBullet physics engine with real-time simulation capabilities
  • • URDF robot model generated from Fusion 360 CAD design
  • • Learning rates: 1e-4 (actor) and 3e-4 (critic) with soft updates

Results & Impact

The project demonstrated the effectiveness of entropy-based RL in robotics, with the SAC implementation significantly outperforming both DQN and traditional approaches. Our Action Replay method proved particularly effective for sim-to-real transfer, achieving stable walking gaits in real-world conditions. The framework provides a foundation for future work in quadrupedal robotics and reinforcement learning applications.
