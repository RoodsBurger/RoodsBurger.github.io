Knolling Bot

Deep Learning YOLO ControlNet Computer Vision

Overview

The Knolling Bot project, conducted under Professor Hod Lipson at the Creative Machines Lab (Columbia University), aims to develop an end-to-end computer vision system that can transform cluttered object arrangements into organized, aesthetically pleasing setups. Building upon previous work in this area, our current iteration focuses on creating a robust pipeline that combines object detection, spatial understanding, and controlled image transformation.

Key Features

  • • YOLO-based object detection system with 86% mAP50 accuracy for precise object identification and matching
  • • ControlNet architecture implementation for image transformation with maintained object integrity
  • • Position-matching algorithm for accurate object correspondence between messy and organized states
  • • Real-time processing capabilities with efficient overlap filtering and spatial analysis

Technical Details

Object Detection System

  • • YOLOv11 nano architecture with pre-trained initialization
  • • Comprehensive data augmentation including rotation, shear, and flips
  • • One-to-one object mapping with position-based validation

ControlNet Implementation

  • • VAE and CLIP text conditioning for precise arrangement control
  • • Perceptual and content loss integration for object preservation
  • • KL divergence term for generation stability

Results & Performance

The system demonstrates robust performance across various metrics. The YOLO-based detection system achieved an mAP50 of 0.86 (±0.02) and mAP50-95 of 0.76 (±0.02), with both precision and recall at 0.84. The ControlNet implementation successfully maintains object integrity during transformation while achieving precise spatial arrangement according to knolling principles.

Previous Work

This project builds upon significant previous work in automated knolling, including two published papers that established the foundational approaches to this problem. While the current implementation represents a new direction in creating an end-to-end solution, it benefits from the insights and methodologies developed in these earlier works.
